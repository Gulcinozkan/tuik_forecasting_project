Forecasting Methods 7.1 Accuracy Measure Function The same accuracy measures were used for all forecasting methods in order to ensure a fair comparison. accuracy_measures <- function(actual, forecast) { error <- actual - forecast bias <- mean(error, na.rm = TRUE) mad <- mean(abs(error), na.rm = TRUE) mse <- mean(error^2, na.rm = TRUE) mape <- mean(abs(error / actual), na.rm = TRUE) * 100 rsfe <- sum(error, na.rm = TRUE) tracking_signal <- rsfe / mad data.frame( Bias = bias, MAD = mad, MSE = mse, MAPE = mape, RSFE = rsfe, Tracking_Signal = tracking_signal ) } 7.2 Naïve Forecasting The naïve forecasting method assumes that the next observation will be equal to the most recent observation. museum_series$naive_forecast <- c( NA, head(museum_series$visitors, -1) ) naive_actual <- museum_series$visitors[-1] naive_forecast <- museum_series$naive_forecast[-1] naive_accuracy <- accuracy_measures( naive_actual, naive_forecast ) naive_next_forecast <- tail( museum_series$visitors, 1 ) naive_accuracy$Method <- "Naive Forecasting" naive_accuracy$Next_Period_Forecast <- naive_next_forecast naive_accuracy ## Bias MAD MSE MAPE RSFE Tracking_Signal ## 1 2081737 7072244 1.316367e+14 28.2546 43716479 6.181416 ## Method Next_Period_Forecast ## 1 Naive Forecasting 61687726 The naïve model produced a MAPE value of approximately 28.25%. Although simple, the method provides a useful benchmark for evaluating more advanced forecasting approaches. 7.3 Moving Average A three-year moving average was selected because the series contains a relatively small number of annual observations and exhibits a long-term trend. museum_series$ma3_forecast <- NA for (i in 4:nrow(museum_series)) { museum_series$ma3_forecast[i] <- mean(museum_series$visitors[(i-3):(i-1)]) } ma_actual <- museum_series$visitors[ !is.na(museum_series$ma3_forecast) ] ma_forecast <- museum_series$ma3_forecast[ !is.na(museum_series$ma3_forecast) ] ma_accuracy <- accuracy_measures( ma_actual, ma_forecast ) ma_next_forecast <- mean( tail(museum_series$visitors, 3) ) ma_accuracy$Method <- "Moving Average (3-Year)" ma_accuracy$Next_Period_Forecast <- ma_next_forecast ma_accuracy ## Selection of the Superior Method Linear Trend Projection was selected as the superior forecasting method. Although several forecasting approaches produced reasonable results, Linear Trend Projection achieved the lowest MAPE value (26.09%) and the lowest MSE among the applicable methods. The selected series displays a clear long-term upward trend and does not contain seasonality. Therefore, a trend-based forecasting model is theoretically appropriate for the structure of the data. The actual versus forecast values also indicate that Linear Trend Projection captures the long-term movement of the series better than the alternative methods. For these reasons, Linear Trend Projection was selected as the superior forecasting method. 10. Interpretation of Results The selected TÜİK variable represents the annual number of museum and archaeological site visitors in Türkiye. The series is suitable for forecasting because it contains a sufficiently long sequence of annual observations and displays a clear trend pattern. The analysis showed that trend-based methods generally performed better than simple averaging methods. The strong decline observed in 2020 reflects the impact of the COVID-19 pandemic, while the subsequent years indicate a substantial recovery in visitor numbers. Linear Trend Projection provided the most appropriate balance between forecasting accuracy and consistency with the observed structure of the series. 12. Limitations Several limitations should be considered. • The series contains only annual observations. • The number of observations is relatively limited. • The COVID-19 pandemic created a major structural break in 2020. • Future visitor numbers may be influenced by economic conditions, tourism policies, and unexpected external events. • The analysis uses only historical visitor counts and does not include explanatory variables. 13.
